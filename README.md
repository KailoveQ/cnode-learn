## 课程介绍
> 本课程旨在从基础入手，循序渐进的深入了解Vue.js的基础知识和进阶内容！
> 
> 本课程分为3个部分，现做主要内容介绍：
## 基础篇
* ---vue.js的基本介绍和概述
* ---数据绑定和指令事件
* ---v-bind指令的深入学习，以及class和style属性的绑定
* ---vue.js的内置指令
* ---表单与v-model
* ---组件
* ---自定义指令
## 进阶篇
* ---render函数的具体使用详解
* ---vue脚手架vue-cli的使用，以及项目搭建的具体介绍
## 实战篇
> 课程学习的最后以一个实战项目高仿CNODE社区作为本次课程的结束。
**项目模块组件：**
* Header模块
* PostList模块
* Article模块
* Slider侧边栏模块
* UserInfo用户个人中心模块
